## 基本ルール

- このフォルダ内のファイルだけを対象にしてください。
- フォルダ外のファイルは読まないでください。
- 変更する前に、何を変更するか短く説明してください。
- 勝手に大きな作り替えをしないでください。
- 必要最小限の変更をしてください。
- 日本語で説明してください。

## HTML / CSS / JavaScript の書き方

- 初心者にも読みやすいコードにしてください。
- 変数名やクラス名は分かりやすくしてください。
- 複雑すぎる書き方は避けてください。
- HTMLは意味が分かるタグを使ってください。
- CSSは整理して書いてください。
- JavaScriptを追加する場合は、何のための処理か説明してください。

## 静的サイト制作のルール

- index.html、style.css、script.js を中心に作業してください。
- 画像を追加する前に確認してください。
- 外部ライブラリを追加する前に確認してください。
- デザインを変更するときは、どこを変えるか説明してください。
- レスポンシブ対応を意識してください。

## 禁止事項

- パスワードやAPIキーを作成・保存しないでください。
- .env ファイルを作らないでください。
- 会社や個人の重要なファイルを読まないでください。
- 許可なく外部サービスと連携しないでください。
- 許可なくファイルを大量に削除しないでください。
